This is a template for starting a project which uses an 8-pin DIP AVR (ATtiny25, ATtiny45, or ATtiny85) with the Arduino programming environment. The template includes the ISP header.

The ATtiny package should be used in the Arduino IDE to provide support for 8-pin AVRs.
