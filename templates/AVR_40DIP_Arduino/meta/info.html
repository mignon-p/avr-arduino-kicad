This is a template for starting a project which uses a 40-pin DIP AVR (ATmega8535, ATmega16, ATmega32, ATmega164, ATmega324, ATmega644, or ATmega1284) with the Arduino programming environment. The template includes the clock crystal, and the ISP header and FTDI (serial) header.

The MightyCore package should be used in the Arduino IDE to provide support for 40-pin AVRs. This template assumes the "Standard" pinout described in the MightCore README.
